### git notes

* allow `latest` tag move:\
  `git config --add remote.origin.fetch "+refs/tags/*:refs/tags/*"`
* prune tags too in prune mode:\
  `git config fetch.pruneTags true`
* enable pruning in standard fetch:\
  `git config fetch.prune true`
